# smsGetter
smsGetter is a project that involves the development of a client Android app, a Flask server, and a Telegram bot. The main purpose of this project is to allow users to receive notifications or SMS on their Android devices, and have those notifications forwarded to a Telegram bot.

The Android client app will be responsible for intercepting the notifications or SMS that come into the user's device, and then sending that data to the Flask server. The server will then process the data and send it to the Telegram bot, which will notify the user about the new message. This process is seamless, and the user can easily stay updated on new messages without having to check their phone constantly.

Overall, the smsGetter project aims to provide a convenient way for Android users to stay informed about their new messages through Telegram. This project is expected to be useful for individuals who receive a lot of notifications or SMS and want to stay informed on the go.
